# Student-Result-Management-System
A simple, console-based C++ application to manage student records. This system allows users to add student details, view all records with calculated percentages, search for specific students, and automatically saves all data to a CSV file for persistent storage.

## Features:

1. Add Student Records: Enter Roll Number, Name, and marks for 5 subjects (English, Maths, Physics, Chemistry, CS).

2. View All Records: Displays a formatted list of all students with individual subject marks, total score, and percentage.

3. Search Functionality: Find specific student details using their Roll Number.

4. Data Persistence: Automatically saves and loads data from students.csv.

5. Single-line Input: Enter all 5 subject marks in a single line for faster data entry.
